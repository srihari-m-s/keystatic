# Product Tech Stack

## Frontend
- **Astro v5**: Modern static site generator for optimal performance
- **TypeScript**: Type-safe development and content structure
- **Tailwind CSS**: Utility-first CSS framework for styling
- **Daisy UI**: Component library built on Tailwind CSS

## Content Management
- **Keystatic**: File-based CMS with UI editing capabilities
- **Markdown**: Content authoring format
- **JSON Schema**: Content structure validation

## Development Tools
- **Node.js**: JavaScript runtime
- **pnpm**: Package manager for efficient dependency management
- **TypeScript**: Type checking and development experience

## Deployment
- **Static Hosting**: Optimized for static site deployment
- **Git**: Version control for content and code
