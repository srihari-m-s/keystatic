# Product Mission

## Pitch
Personal Site with Keystatic CMS is a personal website solution that helps layman users manage their own content by providing a user-friendly content management system built on Keystatic, enabling simple UI-based editing without technical knowledge.

## Users

### Primary Customers
- **Layman Users**: Non-technical individuals who need to manage personal website content without developer assistance
- **Developers**: Technical users who perform one-time setup and handoff to end users

### User Personas
**Content Manager** (25-65)
- **Role:** Personal brand owner, blogger, or small business owner
- **Context:** Managing personal website content regularly without technical skills
- **Pain Points:** Cannot code, needs simple interface to update website content
- **Goals:** Easily update blog posts, edit pages, manage website content independently

## The Problem

### Content Management Complexity
Non-technical users struggle to update website content without developer assistance. This creates dependency on technical support and delays content updates.

**Our Solution:** A Keystatic-powered CMS that provides intuitive UI-based content editing with a simple workflow designed for layman users.

## Differentiators

### Developer-Managed, User-Friendly
Unlike traditional CMS solutions that require ongoing technical maintenance, we provide a one-time developer setup with a simple, intuitive interface for end users. This results in complete content independence for non-technical users.

## Key Features

### Core Features
- **Keystatic CMS Integration:** Full content management system using Keystatic as the backend
- **UI-Based Content Editing:** Intuitive interface for editing all website content without code
- **Simple User Workflow:** Streamlined process designed specifically for non-technical users

### Developer Features
- **One-Time Setup:** Complete initial configuration by developer
- **Handoff-Ready:** System designed for easy transition to end users

### Advanced Features
- **Type-Safe Content:** TypeScript integration ensuring content structure integrity
- **Modern UI Framework:** Responsive design using Tailwind CSS and Daisy UI components
